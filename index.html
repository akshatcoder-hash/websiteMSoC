About

Monsoon Summer of Code (MSoC) is a program designed to bring new, excited contributors into open source communities, with the hope that they will continue to contribute to open source communities. We will provide mentors who act as guides through the entire process, from learning about the community, to helping MSoC Contributors become familiar with the code base and testing practices, to finally releasing their code for the world to use!
